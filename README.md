# Frontend Assignment



## Install
### Install: 
- [node js](https://nodejs.org/en/) (I used version: v10.16.3)
- [yarn](https://classic.yarnpkg.com/en/docs/install#mac-stable) (I used version: 1.17.3)

#### `yarn install`
This project is used `json-server` for api server, `react` as frontend, `redux`, `redux-thunk`, `axios`, `blurhash`,`react-spring`

## Run
In the project directory, you can run:
#### `yarn start`
This will run `mock data server` at `localhost:4000` and `react client`
